# KUTTYBROTHERS Premium Hub

## Project Overview
This is the central management interface for KUTTYBROTHERS operations, featuring a modern "Command Center" dashboard, dynamic theme switching, and modular design archetypes.

## Key Features
- **Command Center Dashboard**: Real-time module filtering and bento-box layout.
- **Visual Customization**: 6 distinct color themes and 5 design archetypes (Glassmorphism, Cyberpunk, etc.).
- **Inventory & Service Management**: Specialized modules for Accounts, Employees, Attendance, Invoices, and Rentals.
- **Modern Tech Stack**: Built with React, TypeScript, Vite, Tailwind CSS, and Framer Motion.

## Development Setup

Follow these steps to run the project locally:

1. **Clone the repository**
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Start the development server**
   ```sh
   npm run dev
   ```

## Technologies Used
- **Vite**: Next-generation frontend tooling.
- **React & TypeScript**: For type-safe component development.
- **shadcn/ui**: Premium UI component library.
- **Framer Motion**: Advanced animations and interactions.
- **Tailwind CSS**: Utility-first styling with custom branding support.
